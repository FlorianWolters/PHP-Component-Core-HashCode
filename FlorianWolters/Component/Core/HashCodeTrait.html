public string hashCode()

Returns a hash code value for this object.

The general contract of {see hashCode} is:

Whenever it is invoked on the same object more than once during an execution of a PHP application, the {see hashCode} method must consistently return the same string. This string need not remain consistent from one execution of an application to another execution of the same application.

As much as is reasonably practical, the {see hashCode} method defined by trait {see HashCodeTrait} does return distinct strings for distinct objects.

The {see hashCode} method implements the Equality Method implementation pattern.
